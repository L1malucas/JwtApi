O que é a Minimal API

A Minimal API é um recurso do .NET que permite criar APIs de forma mais simples e concisa, utilizando uma sintaxe mais enxuta e eficiente. Ela proporciona um desenvolvimento rápido e fácil, reduzindo a quantidade de código necessária para criar uma API.

Autenticação via JWT e Perfis de Usuário

Nossa API utiliza autenticação via JWT (JSON Web Token) para garantir a segurança das informações. Cada usuário possui um perfil associado, que define suas permissões e acessos dentro do sistema. Isso permite um controle granular sobre as funcionalidades disponíveis para cada tipo de usuário.
